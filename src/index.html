proyectos

Find Your DOG!

Pyoyecto final individual del bootcamp de SoyHenry. Una pagina donde podes encontrar razas de perros, con sus características especiales. Los datos se pueden ordenar y filtrar de varios modas, así como hacer búsquedas especificas por nombre de raza. Además se pueden crear razas nuevas, y ver detalles específicos de cada raza. La app esta creada con css puro, sin uso de librerías externas como Bootstrap. Realizada con JavaScript, Redux, React Js, Node JS, PostgressSQL, Sequelize y Express.
